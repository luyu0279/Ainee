QUESTION

Explain the concept of Big O notation in algorithms.

ANSWER

Big O notation describes the upper bound of an algorithm's time complexity or space requirements based on input size. It indicates how runtime/space grows as input size increases:

  • O(1): Constant time (best)
  • O(log n): Logarithmic time
  • O(n): Linear time
  • O(n log n): Log-linear time
